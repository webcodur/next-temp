# Stepper 컴포넌트 수도코드

## 입력값 정의

- 스텝 목록 배열 받기
- 현재 진행된 스텝 번호 받기 (1부터 시작)
- 현재 보고 있는 스텝 번호 받기 (선택사항, 기본값은 현재 진행 스텝)
- 완료된 스텝들 배열 받기 (선택사항)
- 스텝 변경 콜백 함수 받기 (선택사항)

## 메인 로직

### 스텝 클릭 처리 함수

```plain text
함수 handleClick(스텝인덱스):
    스텝번호 = 스텝인덱스 + 1
    
    만약 (스텝번호 <= 현재진행스텝 또는 스텝이완료됨(스텝번호)):
        onChange 콜백 호출(스텝번호)
```

### 스텝 완료 여부 확인 함수

```plain text
함수 isStepCompleted(스텝번호):
    완료된스텝들배열에 스텝번호가 포함되어있는지 반환
```

### 스텝 클릭 가능 여부 확인 함수

```plain text
함수 isStepClickable(스텝번호):
    스텝번호 <= 현재진행스텝 또는 스텝이완료됨(스텝번호) 반환
```

### 스텝 상태 결정 함수

```plain text
함수 getStepState(스텝인덱스):
    스텝번호 = 스텝인덱스 + 1
    
    만약 스텝번호 == 현재보는스텝:
        만약 스텝번호 == 현재진행스텝: 'active' 반환
        만약 스텝이완료됨(스텝번호): 'viewing' 반환
        아니면: 'viewing' 반환
    
    만약 스텝이완료됨(스텝번호): 'completed' 반환
    만약 스텝번호 == 현재진행스텝: 'active' 반환
    만약 스텝번호 == 현재진행스텝 + 1: 'nextAvailable' 반환
    
    아니면: 'inactive' 반환
```

### 스텝 색상 결정 함수

```plain text
함수 getStepColors(스텝인덱스):
    스텝번호 = 스텝인덱스 + 1
    
    만약 스텝번호 == 현재보는스텝 그리고 스텝번호 != 현재진행스텝:
        스텝이완료됨(스텝번호) ? 'text-blue-600' : 'text-blue-500' 반환
    
    만약 스텝이완료됨(스텝번호): 'text-green-600' 반환
    만약 스텝번호 == 현재진행스텝: 'text-primary' 반환
    만약 스텝번호 == 현재진행스텝 + 1: 'text-orange-500' 반환
    
    아니면: 'text-neutral-400' 반환
```

### 스텝 CSS 클래스 결정 함수

```plain text
함수 getStepClasses(스텝인덱스):
    스텝번호 = 스텝인덱스 + 1

            'neu-raised text-blue-600 border-2 border-blue-200' : 
            'neu-raised text-blue-500 border-2 border-blue-100' 반환
    
    만약 스텝이완료됨(스텝번호): 
        'neu-inset text-green-600 hover:neu-raised transition-all cursor-pointer' 반환
    
    만약 스텝번호 == 현재진행스텝: 'neu-raised text-primary' 반환
    만약 스텝번호 == 현재진행스텝 + 1: 'neu-flat text-orange-500 border border-orange-200' 반환
    
    아니면: 'neu-flat text-neutral-400' 반환
```

## 렌더링 로직

### 유효성 검사

```plain text
만약 스텝목록이 없거나 비어있으면:
    null 반환 (아무것도 렌더링하지 않음)
```

### 스텝 목록 렌더링

```plain text
스텝목록 순회하며 각 스텝에 대해:
    스텝 컨테이너 생성:
        - 애니메이션 효과 적용 (순차 등장)
        - 클릭 이벤트 처리
        - 상태별 색상 및 스타일 적용
    
    스텝 원형 버튼 생성:
        - 완료된 스텝: 체크마크 표시
        - 진행 중 스텝: 숫자 표시
        - 특수 효과들:
            * 현재 진행 스텝: 펄스 애니메이션
            * 현재 보는 스텝: 특별 표시 애니메이션
            * 다음 가능 스텝: 힌트 애니메이션
    
    스텝 라벨 생성:
        - 스텝 이름 표시
        - 현재 보는 스텝: 눈 아이콘 추가
        - 다음 가능 스텝: 화살표 아이콘 추가
    
    진행 상태 라인 생성 (마지막 스텝 제외):
        - 기본 회색 라인
        - 완료된 구간: 초록색 라인 애니메이션
        - 진행된 구간: 기본 색상 라인 애니메이션
```

### 네비게이션 헬퍼 렌더링

```plain text
만약 현재보는스텝이 있고 현재진행스텝과 다르면:
    네비게이션 버튼들 표시:
        - "현재 단계로" 버튼 (현재 진행 스텝으로 이동)
        - 다른 완료된 스텝들로 이동하는 작은 버튼들
```

## 애니메이션 정의

### 스텝 상태별 애니메이션

- inactive: 축소, 투명도 감소
- active: 확대, 완전 불투명
- completed: 기본 크기, 완전 불투명
- viewing: 약간 확대, 약간 투명
- nextAvailable: 살짝 확대, 중간 투명도

### 진행 라인 애니메이션

- 왼쪽에서 오른쪽으로 스케일 애니메이션

### 숫자/체크마크 전환 애니메이션

- Y축 회전 효과로 전환
