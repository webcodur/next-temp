# Flow Chart 컴포넌트

React Flow 라이브러리를 기반으로 한 인터랙티브 플로우 차트 다이어그램 컴포넌트입니다.

## 주요 기능

### 기본 조작

- **노드 이동**: 드래그하여 노드 위치 변경
- **줌 제어**: 마우스 휠로 확대/축소
- **패닝**: 우클릭 드래그로 캔버스 이동

### 연결 생성

- **동적 연결**: 노드 모서리에서 드래그하여 다른 노드와 연결
- **애니메이션**: 연결선에 자동 애니메이션 효과 적용
- **스타일링**: 노드별 색상 및 스타일 구분

### 추가 기능

- **미니맵**: 전체 다이어그램 네비게이션
- **컨트롤 패널**: 줌, 핏투뷰, 전체화면 등
- **격자 배경**: 정렬 및 시각적 가이드
- **노드 추가/삭제**: 런타임에 동적 노드 관리

## 기술 스택

- **React Flow**: 핵심 다이어그램 엔진
- **TypeScript**: 타입 안전성
- **Tailwind CSS**: 스타일링

## 사용 사례

- 워크플로우 시각화
- 시스템 아키텍처 다이어그램
- 프로세스 플로우 차트
- 조직도 및 관계도
- 데이터 플로우 다이어그램

## 확장 가능성

- 커스텀 노드 타입 추가
- 다양한 연결선 스타일
- 노드 그룹핑 기능
- 레이어 관리
- 실시간 협업 기능
