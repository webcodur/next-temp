# Next.js 프로젝트

Next.js 기반의 웹 애플리케이션 프로젝트다.

## 실행 환경

Node.js 18 이상이 필요하다.

## 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속한다.

## 빌드

```bash
npm run build
npm start
```

## 환경 변수

프로젝트 루트에 `.env.local` 파일을 생성하여 환경 변수를 설정한다.
